# Tetris Game Project

## Overview
This project is a fully functional implementation of the classic Tetris game. Players manipulate falling blocks to form complete lines and score points.

## Features
- **Single-player Mode**: Play against the clock or try to beat your own high score.
- **Multiple Levels**: As you progress, the game becomes faster and more challenging.
- **Customizable Controls**: Players can set their own keyboard layouts to control the game.
- **Score Tracking**: Track your scores and monitor your progress over time.

## Installation
To run this project locally, follow these steps:
1. **Clone the repository**:
   ```bash
   git clone https://github.com/Ovation-SWE/terminis.git
   ```
2. **Navigate to the project directory**:
   ```bash
   cd terminis
   ```
3. **Install dependencies** (if applicable):
   ```bash
   npm install
   ```

## Usage Instructions
1. **Start the game**:
   ```bash
   npm start
   ```
2. **Control the blocks**:
   - Move left: `Arrow Left`
   - Move right: `Arrow Right`
   - Rotate: `Arrow Up`
   - Soft drop: `Arrow Down`
   - Hard drop: `Space`

3. **Objective**: The goal is to guide falling blocks to complete horizontal lines. When a line is completed, it will disappear, and you will gain points.
